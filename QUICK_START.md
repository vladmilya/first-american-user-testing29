# Quick Start Guide - 5 Minutes to Your First AI-Built Presentation

## Step 1: Open the Application (30 seconds)
1. Navigate to: `file:///Users/vmilyavsky/Documents/Projects/user-testing-synthesis/index.html`
2. The app loads with sample data automatically

## Step 2: Set Up AI Assistant (2 minutes)

### Get an API Key
**Option A: OpenAI (Recommended)**
1. Go to https://platform.openai.com/api-keys
2. Sign up or log in
3. Click "Create new secret key"
4. Copy the key (starts with `sk-...`)

**Option B: Anthropic**
1. Go to https://console.anthropic.com/
2. Sign up or log in
3. Navigate to API Keys
4. Create and copy your key

### Configure the Assistant
1. Click the **purple chat button** (bottom-right corner)
2. Click the **‚öôÔ∏è Settings icon**
3. Paste your API key
4. Select your model (GPT-4 recommended for best quality)
5. Click **Test Connection**
6. Click **Save Settings**

## Step 3: Build Your First Presentation (2 minutes)

### Method 1: Use AI (Easiest)
1. In the chat, type:
   ```
   Create a presentation with key findings and recommendations
   ```
2. Press Enter
3. AI will automatically add slides
4. Navigate to **"Presentation"** in the sidebar to see your presentation
5. Click **"Open Builder"** if you want to make changes

### Method 2: Manual Selection (More Control)
1. Navigate to **"Presentation"** in the left sidebar
2. Click **"Open Builder"** button
3. Use the **dropdown menu** to select sections you want
4. Click **"Add Section"** for each section
5. Reorder slides with ‚Üë‚Üì arrows if needed
6. Click **"View Presentation"**

## Step 4: View & Present (30 seconds)
1. Use **arrow keys** or buttons to navigate slides
2. Press **F** for fullscreen mode
3. Click **üì• Download PDF** to export
